# EtsyPulse Submission Assets

This folder contains written materials for the lablab.ai Web Data UNLOCKED Hackathon submission.

## Files

- `submission-package.md`: main submission copy, market analysis, Bright Data proof, roadmap, and judging criteria mapping.
- `demo-video-script.md`: 2-3 minute demo video script for future Playwright/Remotion capture.
- `slide-outline.md`: slide deck outline for future PDF/deck generation.
- `assets/etsypulse-cover.png`: 16:9 PNG cover image for lablab and README use.
- `assets/etsypulse-cover.svg`: editable source cover image.

## Recommended Submission Values

- Project title: `EtsyPulse: Autonomous Market Briefs for Etsy Sellers`
- Primary track/category: `GTM Intelligence`
- Public demo mode: `DEMO_MODE=true`
- Bright Data proof: dashboard Debug panel, backend `BrightDataClient`, cached real Bright Data-style demo fixtures, live Web Unlocker path, and admin live-smoke endpoint.
